# Система 360-градусной оценки персонала

**Copyright (c) 2025 Стас Чашин для БИТ.Цифра. Все права защищены.**

Комплексная система для проведения 360-градусной оценки персонала с интеграцией Mattermost.

## 🚀 Возможности

### ✅ Реализованные функции

- **Веб-интерфейс**: Удобное управление через браузер с Material-UI
- **База данных**: 7 категорий компетенций, 49 профессиональных вопросов
- **API**: RESTful API для управления пользователями и циклами оценки
- **Интеграция Mattermost**: Готовый сервис для уведомлений и напоминаний
- **Email уведомления**: Красивые HTML письма с результатами оценки
- **Анонимные результаты**: Система защиты конфиденциальности оценивающих
- **Завершение циклов**: Автоматическая отправка результатов участникам
- **Планирование**: Календарное планирование периодических оценок
- **Номинация оценивающих**: Система выбора минимум 5 человек для оценки
- **Система ролей**: Дифференцированный доступ к результатам
- **Визуализация данных**: Интерактивные графики и статистика

### 📊 Категории компетенций

1. **Лидерство** - Оценка лидерских качеств и способности направлять команду
2. **Коммуникация** - Навыки общения и передачи информации
3. **Командная работа** - Способность работать в команде и сотрудничать
4. **Решение проблем** - Аналитические навыки и способность решать сложные задачи
5. **Профессиональное развитие** - Стремление к росту и развитию компетенций
6. **Результативность** - Достижение целей и качество выполнения задач
7. **Адаптивность** - Способность адаптироваться к изменениям

## 🛠 Технологический стек

- **Frontend**: React, Next.js 14, Material-UI, TypeScript
- **Backend**: Next.js API Routes
- **База данных**: SQLite (разработка) / PostgreSQL (продакшн)
- **ORM**: Prisma
- **Интеграция**: Mattermost API
- **Стилизация**: Material-UI с кастомной темой

## 📖 Использование

### Основные страницы

- **/** - Главная страница с быстрыми действиями и статистикой
- **/users** - Управление пользователями (создание, редактирование, удаление)
- **/cycles** - Управление циклами оценки (создание, завершение, мониторинг)
- **/results/[cycleId]** - Просмотр результатов оценки с анонимизацией
- **/reports** - Отчеты и аналитика по оценкам
- **/settings** - Настройки системы и конфигурация
- **/test** - Тестирование API и системы для разработчиков

### API Endpoints

#### Пользователи
- `GET /api/users` - Получить список пользователей
- `POST /api/users` - Создать нового пользователя
- `PUT /api/users/[id]` - Обновить пользователя
- `DELETE /api/users/[id]` - Удалить пользователя

#### Циклы оценки
- `GET /api/cycles` - Получить список циклов оценки
- `POST /api/cycles` - Создать новый цикл оценки
- `PUT /api/cycles/[id]` - Обновить цикл оценки
- `DELETE /api/cycles/[id]` - Удалить цикл оценки

#### Результаты и завершение
- `GET /api/cycles/[id]/results` - Получить результаты цикла с анонимизацией
- `POST /api/cycles/[id]/complete` - Завершить цикл и отправить уведомления

#### Настройки и отчеты
- `GET /api/settings` - Получить настройки системы
- `PUT /api/settings` - Обновить настройки системы
- `GET /api/reports` - Получить аналитические данные

### Управление пользователями

1. Откройте страницу "Пользователи"
2. Нажмите "Добавить пользователя"
3. Заполните форму с данными сотрудника
4. Назначьте менеджера (опционально)

### Создание цикла оценки

1. Откройте страницу "Циклы"
2. Выберите сотрудника для оценки
3. Установите даты начала и окончания
4. Добавьте участников оценки (минимум 5 человек)
5. Запустите цикл

### База данных

#### Основные модели

**User (Пользователи)**
- Хранит информацию о сотрудниках
- Поддерживает иерархию менеджер-подчиненный
- Связан с Mattermost через mattermostUserId

**QuestionCategory → Question (Вопросы)**
- 7 категорий компетенций
- 49 вопросов (7 на каждую категорию)
- Типы: RATING (1-5), TEXT (свободный текст)

**FeedbackCycle (Циклы оценки)**
- Основная сущность для проведения оценки
- Содержит даты, статус, связи с участниками

**CycleParticipant (Участники)**
- Связывает пользователей с циклами
- Определяет роли: SELF, MANAGER, PEER, SUBORDINATE, CUSTOMER

**ReviewerNomination (Номинации)**
- Система выбора оценивающих (минимум 5 человек)
- Статусы: PENDING, APPROVED, REJECTED

**FeedbackResponse (Ответы)**
- Хранит ответы на вопросы
- Поддерживает рейтинги и текстовые ответы

### 🔄 Workflow завершения цикла оценки

После завершения сбора ответов администратор может завершить цикл оценки. Система автоматически:

1. **Проверяет готовность** - показывает процент завершенности оценок
2. **Обновляет статус** - меняет цикл на COMPLETED
3. **Анонимизирует данные** - скрывает имена оценивающих для оцениваемого
4. **Отправляет уведомления** - email и Mattermost всем участникам
5. **Предоставляет доступ** - к результатам с разными уровнями детализации

### 🔒 Система анонимизации и прав доступа

Система обеспечивает разные уровни доступа к результатам оценки:

| Роль | Доступ к данным | Видимость оценивающих |
|------|-----------------|----------------------|
| 👤 **Оцениваемый** | Результаты своей оценки | ❌ Анонимно |
| 👔 **Руководитель** | Результаты подчиненных | ✅ Полные имена |
| 👨‍💼 **Администратор** | Все результаты | ✅ Полные имена |

**Примеры анонимизации:**
- **Для оцениваемого**: "Коллега А", "Коллега Б", "Руководитель"
- **Для руководителя**: "Петров И.И.", "Сидорова А.С.", "Иванов В.В."
